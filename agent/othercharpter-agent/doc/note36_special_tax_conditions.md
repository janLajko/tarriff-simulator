# Note 36 - 特殊課稅條件表格

> **文件來源**: note36_Chapter 99_2025HTSRev32_.pdf
> **主題**: 半成品銅及銅密集衍生品 (Semi-finished Copper and Intensive Copper Derivative Products)

## 一、稅目架構總覽

### 基礎稅目與 Subdivision 對應

| 稅目 | 適用 Subdivision | 產品類別 | 說明 |
|-----|-----------------|---------|------|
| 9903.78.01 | (b) | 銅含量部分 | 對銅製品的**銅含量價值**課徵從價稅 |
| 9903.78.02 | (b) | 非銅含量部分 | 對銅製品的**非銅含量價值**課徵 |

### 關係類型說明

| 關係類型 | 稅目 | 說明 |
|---------|------|------|
| **配套關係** | 9903.78.01 + 9903.78.02 | 同一批貨物拆成兩部分課稅 |

> **Note 36 沒有「潛在替代關係」**，只有「配套關係」和「特殊課稅規則」。

---

## 二、配套稅目關係說明

### 9903.78.01 與 9903.78.02 的配對關係

當進口 subdivision (b) 列出的銅製品時，需將貨物價值拆分為兩部分分別課稅：

```
┌─────────────────────────────────────────────────────────┐
│  銅製品總價值 (Total Value)                              │
│  ┌─────────────────────┐  ┌───────────────────────────┐ │
│  │  銅含量價值          │  │  非銅含量價值             │ │
│  │  (Copper Content)    │  │  (Non-Copper Content)     │ │
│  │                      │  │                           │ │
│  │  → 適用 9903.78.01   │  │  → 適用 9903.78.02        │ │
│  │    (以 kg 申報)      │  │                           │ │
│  └─────────────────────┘  └───────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

**配對原因**：
- 9903.78.01：處理銅製品中「銅含量」的價值部分
- 9903.78.02：處理銅製品中「非銅含量」的價值部分
- 兩者必須**同時使用**，針對同一批貨物的不同價值組成

**申報流程**：
1. 計算產品中的銅含量重量 (kg)
2. 計算銅含量的申報價值
3. 9903.78.01 從價稅僅對銅含量價值課徵
4. 非銅含量部分依 9903.78.02 處理

## 三、特殊課稅規則總覽

| # | 段落位置 | 適用稅則 | 特殊條件名稱 | 詳細說明 |
|---|---------|---------|-------------|---------|
| 1 | Note 36(a) 第2段 | 9903.78.01 | **關稅堆疊 (FTA + 銅稅)** | 符合 FTA 或優惠計畫資格的產品，9903.78.01 的稅率將**額外加徵**於原本適用的特殊稅率之上。<br><br>關鍵句："the duty provided in heading 9903.78.01 shall be collected **in addition to** any special rate of duty otherwise applicable" |
| 2 | Note 36(a) 第2段 | 9802.00.60 | **全額價值課稅** | 依 Chapter 98 條款申報入境的貨物，9802.00.60 項下的關稅應以進口物品的**完整價值** (full value) 評估，而非僅針對增值部分。 |
| 3 | Note 36(a) 第3段 | 9903.78.01 | **銅含量計算** | 9903.78.01 的從價稅**只適用於銅含量的申報價值**，而非整件產品價值。<br><br>**申報要求**：銅含量需以 **kg** 為單位申報，作為 9903.78.01 的數量單位，同時仍需依第1-97章規定申報產品本身的數量單位。 |
| 4 | Note 36(a) 第2段 | 9903.78.01 | **AD/CVD 照加** | 反傾銷稅 (antidumping)、反補貼稅 (countervailing) 或其他適用的稅費，**仍需加徵**於 9903.78.01 之上。 |
| 5 | Note 36(a) 第2段 | 9903.78.01 | **Subchapter III/IV 額外稅照加** | 即使禁止適用較低稅率，Chapter 99 subchapter III 或 subchapter IV 中規定的**額外關稅仍需加徵**。 |
| 6 | Note 36(c) | 9903.78.02 | **非銅含量適用** | 9903.78.02 適用於 subdivision (b) 列出產品的**非銅含量部分**。 |
| 7 | Note 36(d) | 9903.78.01, 9903.78.02 | **資訊申報義務** | 進口商需提供 CBP 認為必要的任何資訊，以便管理這些稅則。 |

## 四、關稅堆疊示意

| 情境 | 計算方式 |
|-----|---------|
| 產品符合 FTA 優惠 | FTA 特殊稅率 **+** 9903.78.01 銅稅 |
| 產品有 AD/CVD | 一般稅率 **+** 9903.78.01 銅稅 **+** AD/CVD |

## 五、9802.00.60 全額價值規則

| 一般情況 | Note 36 規定 |
|---------|-------------|
| 依 9802.00.60 可能只對增值部分課稅 | 必須以**完整價值**課稅 |

## 六、名詞解釋

| 術語 | 英文 | 說明 |
|-----|------|------|
| 半成品銅 | Semi-finished Copper | 經過初步加工但尚未製成最終產品的銅材 |
| 銅密集衍生品 | Intensive Copper Derivative Products | 銅含量較高的衍生製品 |
| 從價稅 | Ad Valorem | 按貨物價值百分比計算的關稅 |
| 銅含量 | Copper Content | 產品中銅的重量及其對應價值 |
| 反傾銷稅 | Antidumping Duty (AD) | 針對以低於正常價值銷售的進口產品課徵的額外關稅 |
| 反補貼稅 | Countervailing Duty (CVD) | 針對受到政府補貼的進口產品課徵的額外關稅 |
| CBP | U.S. Customs and Border Protection | 美國海關及邊境保護局 |
| 配套關係 | Paired Relationship | 兩個稅目同時適用於同一批貨物的不同部分 |
