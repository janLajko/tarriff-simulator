## 核心概念：什麼是「潛在替代關係」？

**替代 (Substitution)**：同一批商品（共用同一個 HTS code 範圍 / subdivision）因為**不同條件**，選擇適用不同的 Chapter 99 稅目。

> 與「互斥」不同：互斥是不同商品類別之間的豁免關係；替代是同一批商品因條件不同而選擇不同稅目。

## 四、配套稅目關係說明

### 9903.74.03 與 9903.74.06 的配對關係

當 USMCA 車輛經商務部批准後，一輛車會被拆分成兩部分計稅：

```
┌─────────────────────────────────────────────────────────────┐
│  車輛總價值 (Total Value)                                    │
│  ┌─────────────────────────┐  ┌───────────────────────────┐ │
│  │  美國含量               │  │  非美國含量                │ │
│  │  (U.S. Content)         │  │  (Non-U.S. Content)        │ │
│  │                         │  │                            │ │
│  │  → 適用 9903.74.06      │  │  → 適用 9903.74.03         │ │
│  │  （較優惠稅率）          │  │  （課徵從價稅）            │ │
│  └─────────────────────────┘  └───────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

**配對原因**：
- 9903.74.03：處理車輛中「非美國含量」的價值部分
- 9903.74.06：處理車輛中「美國含量」的價值部分
- 兩者必須**同時使用**，針對同一輛車的不同價值組成

## 六、9903.74.09 的排除規則

Note 38(j) 規定 9903.74.09 **不適用於**以下物品：

| 排除項目 | 說明 |
|---------|------|
| HTSUS 第 72 章 | 鐵及鋼 |
| HTSUS 第 73 章 | 鐵或鋼製品 |
| HTSUS 第 76 章 | 鋁及其製品 |
| subdivision (i) 列舉物品 | 已明確列在 9903.74.08 清單中的零件 |
| U.S. note 33(g) 列舉物品 | 另一個 note 中定義的特定物品 |

## 七、名詞解釋

| 術語 | 英文 | 說明 |
|-----|------|------|
| 美國含量 | U.S. Content | 可歸因於支持國內生產的美國本土活動的價值 |
| 非美國含量 | Non-U.S. Content | 車輛總價值減去美國含量 |
| 從價稅 | Ad Valorem | 按貨物價值百分比計算的關稅 |
| 完全拆解套件 | Knock-down Kit | 完全拆解的車輛，需組裝才能成為完整車輛 |
| 零件組合包 | Parts Compilation | 多個零件打包成的組合 |
| USMCA | United States-Mexico-Canada Agreement | 美墨加協定（取代 NAFTA） |
| 安全網條款 | Safety Net Provision | 處理 HTS 歸類與實際物品狀態不符的例外條款 |
