## 核心概念：什麼是「潛在替代關係」？

**替代 (Substitution)**：同一批商品（共用同一個 HTS code 範圍 / subdivision）因為**不同條件**，選擇適用不同的 Chapter 99 稅目。

Note 37 的替代關係分為兩種類型：
1. **國家條件替代**：根據來源國選擇不同稅目
2. **狀態條件替代（豁免）**：根據產品是否為成品選擇不同稅目

## 四、關稅處理方式說明

### 堆疊關係 (In Addition To)

**適用稅目**：9903.76.01, 9903.76.02, 9903.76.03, 9903.76.20

**定義**：當產品符合 FTA 或優惠計畫時，Chapter 99 關稅**額外加上**原有優惠稅率。

```
總關稅 = FTA 優惠稅率 + 9903.76.xx 稅率
```

**原文關鍵句**：
> "the duty provided in heading 9903.76.xx shall be collected **in addition to** any special rate of duty otherwise applicable under the appropriate tariff subheading"

### 取代關係 (In Lieu Of)

**適用稅目**：9903.76.21, 9903.76.22, 9903.76.23

**定義**：當產品符合 FTA 或優惠計畫時，Chapter 99 關稅**取代**原有優惠稅率（但 AD/CVD 等其他稅仍照加）。

```
總關稅 = 9903.76.xx 稅率（FTA 優惠不適用）
```

**原文關鍵句**：
> "the duty provided in heading 9903.76.xx shall be collected **in lieu of** any special rate of duty otherwise applicable under the appropriate tariff subheading"

### 美韓 FTA 例外 (9903.76.23)

南韓產品適用 9903.76.23 的「取代關係」，但有例外：

> "except for goods qualifying under the **United States-Korea Free Trade Agreement**"

符合美韓 FTA 的產品可繼續使用 FTA 優惠稅率。

## 六、名詞解釋

|-----|------|------|
| 替代關係 | Substitution | 同一 HTS code 範圍，因條件不同選擇不同稅目 |
| 國家條件替代 | Country-based Substitution | 根據來源國選擇適用的稅目 |
| 狀態條件替代 | Status-based Substitution | 根據產品狀態（成品/非成品）選擇適用的稅目 |
| 堆疊關係 | In addition to | 額外加徵，原有優惠稅率仍適用 |
| 取代關係 | In lieu of | 取代原有優惠稅率 |
| 豁免 | Exemption | 免除額外關稅 |
| 軟木木材 | Softwood Timber/Lumber | 軟木（如松木、雲杉）的原木及加工木材 |
| 軟墊木製家具 | Upholstered Wooden Furniture | 有軟墊的木製家具，如沙發、椅子 |
| 廚房櫥櫃 | Kitchen Cabinet | 廚房用的儲物櫃 |
| 梳妝台 | Vanity | 浴室或臥室的洗手台/化妝台 |
| 成品 | Completed | 完整的成品 |
| 非成品 | Non-completed | 半成品或未完成的產品 |
| AD/CVD | Antidumping/Countervailing Duties | 反傾銷稅/平衡稅 |
| FTA | Free Trade Agreement | 自由貿易協定 |
| 美韓 FTA | US-Korea Free Trade Agreement | 美國-韓國自由貿易協定（KORUS FTA） |
| ISO 代碼 | ISO Country Code | 國際標準化組織的國家代碼（ISO 3166-1 alpha-2） |

## 七、全額價值課稅規則 (Full Value)

### 定義
若依據 Chapter 98 條款申報入境（特別是 9802.00.60），關稅應以進口物品的**完整價值**評估，而非僅針對增值部分。

### 適用範圍
Note 37 中所有主要稅目都適用此規則：

| 稅目 | 段落位置 |
|-----|---------|
| 9903.76.01 | Note 37(a) |
| 9903.76.02 | Note 37(c) |
| 9903.76.03 | Note 37(e) |
| 9903.76.20 | Note 37(h) |
| 9903.76.21 | Note 37(i) |
| 9903.76.22 | Note 37(j) |
| 9903.76.23 | Note 37(l) |

### 原文關鍵句
> "duties under subheading 9802.00.60 shall be assessed based upon the **full value** of the imported article"
