## 核心概念：什麼是「潛在替代關係」？

**替代 (Substitution)**：同一批商品（共用同一個 HTS code 範圍 / subdivision）因為**不同條件**，選擇適用不同的 Chapter 99 稅目。

> 與「互斥」不同：互斥是不同商品類別之間的豁免關係；替代是同一批商品因條件不同而選擇不同稅目。

## 四、配套稅目關係

### 9903.94.02 + 9903.94.03 的配對關係

當 USMCA 車輛經商務部批准後，一輛車會被拆分成兩部分計稅：

```
┌─────────────────────────────────────────────────────────────┐
│  車輛總價值 (Total Value)                                    │
│  ┌─────────────────────────┐  ┌───────────────────────────┐ │
│  │  美國含量               │  │  非美國含量                │ │
│  │  (U.S. Content)         │  │  (Non-U.S. Content)        │ │
│  │                         │  │                            │ │
│  │  → 適用 9903.94.02      │  │  → 適用 9903.94.03         │ │
│  │    （較優惠稅率）        │  │    （課徵 25% 從價稅）      │ │
│  └─────────────────────────┘  └───────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

**配對原因**：
- 9903.94.02：處理車輛中「美國含量」的價值部分（較優惠稅率）
- 9903.94.03：處理車輛中「非美國含量」的價值部分（課徵 25% 從價稅）
- 兩者必須**同時使用**，針對同一輛車的不同價值組成

**計算流程**：
1. 進口商向商務部 (Secretary of Commerce) 提交文件
2. 說明車輛中的美國含量價值
3. 商務部審核批准
4. 美國含量部分適用 9903.94.02
5. 非美國含量部分課徵 25% 關稅 (9903.94.03)

### AD/CVD 堆疊規則

所有適用稅則均明確規定：
> "All antidumping, countervailing, or other duties and charges applicable to such goods shall continue to be imposed **in addition to** the duty in heading 9903.94.xx"

**意義**：反傾銷稅 (AD)、平衡稅 (CVD) 及其他稅費**一律加上** 9903.94.xx 的關稅，無例外。

## 九、從價稅等值計算 (Ad Valorem Equivalent)

| 段落位置 | 適用對象 | 計算方式 |
|---------|---------|---------|
| Note 33(m) | 日本、歐盟、韓國產品（適用特定稅或複合稅率者） | **公式**：從價稅等值 = 應付稅額 ÷ 關稅價值 × 100% |

**範例**：
- 若某商品適用特定稅 50 美分/公斤，進口 1 公斤，關稅價值 $10
- → 從價稅等值 = $0.50 ÷ $10 = 5%

**韓國特例**：若正確申報 column 1-Special 下的特定稅或複合稅率，也以相同方式計算

## 十二、名詞解釋

| 術語 | 英文 | 說明 |
|-----|------|------|
| 美國含量 | U.S. Content | 可歸因於在美國境內完全獲得、完全生產或實質轉型的零件價值 |
| 非美國含量 | Non-U.S. Content | 車輛總價值減去美國含量價值 |
| 從價稅 | Ad Valorem | 按貨物價值百分比計算的關稅 |
| 特定稅 | Specific Duty | 按貨物數量/重量計算的固定金額關稅 |
| 複合稅 | Compound Duty | 從價稅與特定稅的組合 |
| 完全拆解套件 | Knock-down Kit | 完全拆解的車輛，需組裝才能成為完整車輛 |
| 零件組合包 | Parts Compilation | 多個零件打包成的組合 |
| 最終用途認證 | End-use Certification | 進口商聲明貨物的預期用途 |
| USMCA | United States-Mexico-Canada Agreement | 美墨加協定（取代 NAFTA） |
| 關稅配額 | Tariff Rate Quota (TRQ) | 配額內適用較低稅率，超過配額適用較高稅率 |
| 特許外國身分 | Privileged Foreign Status | 外貿區貨物的一種身分，入境消費時適用入區時的稅率 |
| 國內身分 | Domestic Status | 外貿區中已完稅的貨物身分 |
| 反傾銷稅 | Antidumping Duty (AD) | 針對以低於正常價值銷售的進口商品課徵的額外關稅 |
| 平衡稅 | Countervailing Duty (CVD) | 針對受到外國政府補貼的進口商品課徵的額外關稅 |
| 製造商進口調整抵消額 | Manufacturer's Import Adjustment Offset Amount | 商務部核定的可抵減金額 |
